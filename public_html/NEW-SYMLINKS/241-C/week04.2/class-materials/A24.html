Working with Colors

Colors are an important part of how Web sites are perceived.  There are different ways to denote the colors with the code that you write.

Lesson Objective:  Background images and colors

 

Hexadecimal Color Values

When working with colors, often we use hexadecimal (hex for short) value pairs to denote colors.  These are six sets of characters represented by #RRGGBB for Red Green And Blue.  The numbers are values between 0 and 255 and are represented in hex from 00 to FF.
