Understanding The Basic Syntax of CSS

CSS is simply a set of rules being applied to your Web page content.  Think of each grouping as a "rule".  Style has a very simple beginning syntax.

 

body {
     background-color: teal;
     color: white;
}
h1 {
     color: yellow;
     font-style: italic;
}

 

In the examples above, each piece has a name and a definition.

Example Name Definition
body Selector the element, class of elements, or unique element identifier that is going to be changed by the rule being defined. 
background-color Property the part of the body that is being changed by the rule
teal Value the value being applied to the property
color Property the part of the body that is being changed by the rule
white Value the value being applied to the property

 

In the example above, can you identify the second selector?  The first property being changed of the second selector? The value being assigned to the second first property of the second selector?

 

Adding CSS to your HTML Pages

There are three ways that you can add style to your pages.  These three means of adding style are:

  • inline - within an actual HTML element
  • embedded - within a <style> element
  • external - link to an external css file

It is important to know the order of these as inline has the highest precedence within your Web page.  If you inline style rule is the same as something that you've created in your embedded style, the inline style will rule will be rendered.  Likewise, if you have a rule in your embedded style that is the something you've defined in your external style sheet, the embedded style will be rendered. 

Inline Style

Inline style is created within an actual HTML element.  Example:

<!DOCTYPE html>
<html lang="en">
<head>
     <title>Inline Style Example</title>
     <meta charset="utf-8"/>
     <meta name="author" content="Marti Baker" />
     <meta name="date" content="11/12/2013" />
     <meta name="robots" content="NOINDEX, NOFOLLOW " />
</script>
</head>
<body>

<!-- Your visible page content -->
<p>This a paragraph about nothing.  There is no style applied to this paragraph</p>

<p style="background-color: #66CCCC;">This is a paragraph about something.  The background color of this paragraph is light blue.</p>

</body>
</html>

 

Embedded Style

Embedded style is created within the head of your Web document.  Example:

<!DOCTYPE html>
<html lang="en">
<head>
     <title>Embedded Style Example</title>
     <meta charset="utf-8"/>
     <meta name="author" content="Marti Baker" />
     <meta name="date" content="11/13/2013" />
     <meta name="robots" content="NOINDEX, NOFOLLOW " />
     <style type="text/css">
         h1 {
             color: yellow;
             font-style: italic;
         }
     </style>
</script>
</head>
<body>
<!-- Your visible page content -->
<h1>Attention World!</h1>

<p>The heading above would be rendered in yellow and italic when displayed in a browser.</p>

</body>
</html>

 

External Style Sheets

External style sheets are used to control the full look and feel of a complete Web site.  They are efficient and effective.  In the first example, I show you how an external style sheet, including comments are created.  External style sheets are saved with a .css file extension.  In the second example, I show you how to reference external style sheet in your Web page.

/* These are style sheet comments */
/* Be sure to include your name within your comments */

body {
    background-color:teal;
    color:white;
    font-family:verdana;

}
h2 {
    text-align:center;
}

 

 

<!DOCTYPE html>
<html lang="en">
<head>
     <title>Including External Style Sheets</title>
     <meta charset="utf-8"/>
     <meta name="author" content="Marti Baker" />
     <meta name="date" content="11/12/2013" />
     <meta name="robots" content="NOINDEX, NOFOLLOW " />
     <link rel="stylesheet" type="text/css" href="includes/mystyle.css"/>
</script>
</head>
<body>
<!-- Your visible page content -->

<h2>Hello World!</h2>
<p>Welcome to a page which will have a teal background, the Hello World! above will be centered, and the text will all be in white, and the font will be Verdana.</p>
</body>
</html>
