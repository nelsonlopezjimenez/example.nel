HTML elements and tags

HTML 5 is an upgrade to older versions of of HTML; html 4.01 and XHTML 1.0.  Understanding the basics of HTML will get you going in the right direction.

Lesson Objective:  Use HTML elements and tags to format paragraphs and text.

 

Container and Empty Elements

A concept that isn't generally talked about any longer but I still feel is important to understand that HTML elements fall into two different groups.  These groups are container elements and empty elements. Container elements include a starting tag, content, and a closing tag.  Empty elements only consist of a starting tag.

Container Elements

Container elements can include either text or other elements.  Examples of common container elements include:

  • <html>
  • <head>
  • <title>
  • <body>
  • <div>
  • <p>
  • <em>

Empty Elements

Empty elements stand alone, they usually insert something into your Web page, such as a line break, a horizontal rule, an image, or a piece of multimedia.  Examples of empty elements include:
  • <img>
  • <hr>
  • <embed>
  • <br>

Required HTML5 Elements

Although HTML5 is much loser than previous versions of HTML, there are still a few minimum elements.  Here is the basic template for you to work from:


<!DOCTYPE html>
<html lang="en">
<head>
     <title>A Good Page Title Here</title>
     <meta charset="utf-8"/>
     <meta name="author" content="your name">
     <meta name="date" content="date created">
     <meta name="robots" content="NOINDEX, NOFOLLOW" />
</head>
<body>
<!-- Your visible page content -->
</body>
</html>

 

Scroll over the template above, save it as template.htm or template.html.  Be sure to remember to change the "your name" to your real name, and "date created" to the date that you complete any assignment.
