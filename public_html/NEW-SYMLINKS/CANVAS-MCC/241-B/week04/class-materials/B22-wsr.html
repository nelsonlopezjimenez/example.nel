CSS and Graphical Elements

Cascading Style Sheets (CSS) and graphical elements of Web pages / sites allow Web developers to create visual effects and determine how pages will be displayed within Web browsers

HTML5 Coding

At the end of this lesson, you should be able to:

  • Incorporate graphical images into HTML pages.
  • Distinguish among and identify the uses and benefits of various graphic file formats, including GIF, GIF89a, JPEG, PNG, TIFF, BMP.
  • Add tiled images and colors to Web page backgrounds.
  • Insert horizontal rules into Web pages.
  • Define the browser-safe color palette.
  • Identify ways that color affects the principles of line, value, shape and form in Web pages.
  • Identify and demonstrate the impact of color combinations to various audiences and cultures.
  • Evaluate Web page design and layout.
  • Explain how to structure Web documents with CSS.
  • Identify ways to apply styles with CSS.
  • Use CSS and HTML5 elements to create document structure.
  • Distinguish between fixed-width and liquid design layouts.
