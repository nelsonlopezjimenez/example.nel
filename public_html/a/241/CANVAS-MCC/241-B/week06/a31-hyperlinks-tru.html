Hyperlinks

Hyperlinks are like portals, as one student said. Portal to different dimensions. In Harry Potter's world, it is how they travel from one place to a different place.

Hyperlinks allow us to investigate what interests most on a page.

At the end of the lesson, you should be able to:

  • Create HTML hyperlinks for text, images, local files and remote sites (internal and external links).
  • Manage existing sites (e.g., remove dead links and/or upgrade connectivity when necessary).
  • Identify the importance of online indexing and cataloging.
