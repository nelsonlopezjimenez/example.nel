Web Development I

  1. Perform content design and technical analysis on web applications and websites.
  2. Use HTML5 to develop, debug, maintain, and document web applications and websites.
  3. Compare and contrast different browsers’ effects on HTML5 documents.

Welcome Week 9

Responsive Design

PLAN At the end of this lesson, you should be able to:

  • The ever-evolving browser and device landscape
  • Defining responsive web design
  • Setting browser support levels
  • A brief discussion on development tools and text editors
  • Our first responsive example: a simple HTML5 page
  • The viewport meta tag
  • Fluid images
  • The Essentials of Responsive Web Design
  • Writing CSS3 media queries to make pages adapt
  • The shortfalls in our basic example
  • Sectioning, grouping, and text-level HTML elements
  • Semantic elements

To demonstrate your competency with these topics you will:

  • Create a simple Responsive Web page incorporating:
    1. Sectioning elements: main, header, article, aside, footer, sidebar areas
    2. Grouping elements: paragraphs, blockquotes, figure, figcaption, details, summary
    3. Text-level semantic: previous inline elements
    4. Embedding media
    5. HTML5 syntax validation
  • Create semantically rich and meaningful code
  • Ensure pages are meaningful and usable with assistive technologies

Welcome Week 05

HTML5 coding

PLAN At the end of this lesson, you should be able to:

  • Use HTML elements and tags to format paragraphs and text.
  • Add comments to HTML code and document page/site creation.
  • Explain the importance of consistently developing to a single W3C standard.
  • Validate Web page design according to technical and audience standards adopted by employers.
  • Create a Web page using the HTML5 standard.
  • Test and validate Web documents.
  • Determine and create appropriate site folder structure.

To demonstrate your competency with these topics you will:

  • Create a simple Web page incorporating:
    • DOCTYPE declaration asp
    • DOCTYPE declaration html
    • <html> tag
    • <head> tag
    • <meta> tag
    • <title> tag
    • <body> tag
    • <table> tag
    • Paragraph formatting and block-level elements
    • Heading elements
    • Text-Level Elements
    • Lists
      • Unordered Lists
      • Ordered Lists
    • HTML comments
    • Box Model
    • Validate a Web document the W3C validator
    • Select and create a site structure for module file project
