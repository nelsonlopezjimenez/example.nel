Web Development II

Course-level Learning Objectives: CIS 242.

  • Use object-oriented client-side scripting with well-formed web pages.
  • Recognize client-side variables and data types and the operations that can be performed on them.
  • Write client-side functions, event handlers and control structures.
  • Verify form data through scripting validation.
  • Save state information through the use of hidden form fields, query-strings and cookies.
  • List the concepts of server-side programming and Node.js.

OUTCOMES

  1. Identify and explain the basic logic structures used in computer programming.
  2. Create and follow a programming bubble chart.
  3. Create and follow flow charts of various logic structures.
  4. Create form that satisfies limited user requirements.
  5. Discern when to use a branching structure in logic development.
  6. Discern when to use a looping structure in logic development.
  7. Be able to identify and use the more common JavaScript controls and their properties.
  8. Use JavaScript to produce a complete program starting with a program requirements statement.
